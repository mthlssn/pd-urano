Gerenciador financeiro

O Gerenciador Financeiro é uma aplicação no formato Web que as empresas podem fazer a integração de dados de sistemas internos de negócios, como Contas a Pagar, Contas a Receber e Folha de Pagamentos, fazendo a entrada de dados por transação customizada aos padrões de documento (Ex. boletos, tributos, TED etc.) ou através de importação de dados via arquivo padrão no Layout FEBRABAN.

Intercâmbio padrão FEBRABAN

O conjunto de dados é equivalente ao módulo “Intercâmbio Padrão FEBRABAN”, com a distinção de poder executar diretamente o processo de importação, com a validação dos dados sendo feita de forma imediata.

Customizável

Nos processos de confirmação de envio de dados para submissão aos sistemas de negócios, podem ser feitos ajustes nos dados para posterior confirmação. O usuário de aprovação pode ter autorização superior, permitindo diferenciar quem opera as transações (Entrada dos dados) e quem autoriza o processamento dos dados (Aprovação).
